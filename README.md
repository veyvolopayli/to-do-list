# To-Do List App

Это веб-приложение для управления задачами, разработанное в рамках лабораторной работы по TDD и CI/CD.

## Функционал
- Создание задач
- Просмотр списка задач
- Обновление статуса и удаление задач
- Автоматическая генерация документации API

## Установка и запуск
1. `docker-compose up --build`
2. Открыть http://localhost:5000

## Документация
Документация к коду генерируется автоматически в CI/CD пайплайне с помощью pdoc.
